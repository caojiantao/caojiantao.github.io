## 1 准备条件

GitHub 账号；

## 2 创建项目

新建一个项目，项目名格式为`${username}.github.io`, username 为你的 github 账号名；

![](http://pics.caojiantao.site/9f5ce62f9d8199b0de8c4fdd1adf32ca.webp)

## 3 选择主题

进入项目设置，选择博客主题；

![](http://pics.caojiantao.site/88e7ae31e2735498246364aafd0b2f8c.webp)

> 这里我选择了 Cayman 主题，简单朴素支持响应式布局。

## 4 设置首页

在项目根目录创建 index.md，添加如下内容；

```markdown
欢迎进入我的在线博客。
```

## 5 在线验证

访问`https://${username}.github.io`，验证成果；

![](http://pics.caojiantao.site/96c495147161027c5ce8f3f29eace3d1.webp)
